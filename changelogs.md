## ✨ Nuevas funciones del server

### ☆ Sistema de perfiles interactivo *(comando `/profile`)*
- Visualización de datos del usuario *(monedas, nombre, progreso, etc.)*  
- Base para futuras mecánicas del servidor  

### ☆ Sistema de **achivements & badges**
- Rangos progresivos: *(new, epic, legendary, mythic, master)*  
- Desbloqueo según actividad, experiencia y participación  

### ☆ Sistema de búsqueda de equipo *(comando `/team`)*
- Permite hacer ping al rol de team *(opcional)*  
- Optimizado para partidas diarias y torneos  

### ☆ Sistema de soporte completo
- Menú de navegación interactivo  
- Canal dedicado: `#support-guidelines` *(reglas y uso del soporte)*  

### ☆ Sistema de tickets
- Atención organizada para dudas, problemas y reportes  

### ☆ Sistema anti-inactividad
- Rol automático para usuarios inactivos *(tras 2 semanas)*  
- Posibilidad de reactivación mediante interacción  

### ☆ Sistema de bienvenida por DM
- Mensaje automático a nuevos miembros  
- Incluye links, guía rápida e invitación a compartir el server  

### ☆ Sistema de torneos y eventos
- Torneos organizados con reglas claras  
- Eventos únicos para la comunidad de Sigmally  
- Ejemplo: *“Mejor jugada del año”* con recompensas  

### ☆ Espacio para creadores de contenido
- Canal dedicado: `#clips-gameplay`  
- Incentivos para contenido viral y participación  

### ☆ Sección de recursos para jugadores
- Skins, macros y herramientas útiles  
- Integración futura con página externa  

### ☆ Página de guía/documentación
- Contenido útil para jugadores nuevos y avanzados  
- Complemento externo del servidor  

### ☆ Sección de preguntas frecuentes *(FAQ)*
- Sobre el server y sobre Sigmally  
- También disponible en la página externa  

### ☆ Integración con Wikipedia *(wiki-bot)*
- Búsqueda rápida desde el servidor  

### ☆ Sistema musical mejorado
- Embeds más completos *(info adicional, recomendaciones, etc.)*  

### ☆ Sistema exclusivo para miembros del clan
- Canales y utilidades privadas  
- Mayor sentido de pertenencia  

### ☆ Server Navigation
- Guía interna del servidor *(canales, uso, info relevante)*  

### ☆ Repositorio del server en GitHub
- Mejor organización del contenido  
- Gestión más profesional del proyecto  

### ☆ Nuevo modo de juego exclusivo
- Creado por Dani! *(especial del server)*  

### ☆ Nuevos comandos personalizados
- Expansión del sistema del bot  
- Más utilidades y funciones internas
